# Number Guessing Game (Python)

This is a simple Python number guessing game.

## How it works
- The computer randomly selects a number between 1 and 50.
- The user keeps guessing until the correct number is found.
- The program shows hints to guess higher or lower.
- It also counts the total number of attempts.

## How to run
1. Make sure Python is installed.
2. Run the program using:
   python Guess_Num.py

## Concepts Used
- Random module
- While loop
- Conditional statements
- User input
